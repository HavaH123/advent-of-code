## What is this?
Solutions to advent calendar type daily coding questions on [adventofcode.com](https://adventofcode.com)

## How to run
Every year has 25 days of adventofcode  (2 questions per day)

### Part 1
```
$ cd {year}/{day}-1
$ node code.js
```

### Part 2
```
$ cd {year}/{day}-2
$ node code.js
```
